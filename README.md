# Cafe Studio

Personal desktop operations app for managing Gloria Jean's Coffee South Africa administrative workflows.

## Purpose

Reduce repetitive month-end administration and reporting.

This is not an enterprise ERP system.
This is a single-user desktop application used internally by one administrator.

## Core Objectives

- Track monthly tasks
- Store operational data
- Generate reports
- Centralise franchise reporting
- Reduce spreadsheet dependency

## Stores

### Head Office
- Oceans Mall
- Express Stores

### Franchise
- Gateway
- Florida Road
- Pavilion
- Lakefield
- Point Waterfront

## Australia Reporting

Include:
- Oceans Mall
- Gateway
- Florida Road
- Pavilion
- Lakefield

Exclude:
- Point Waterfront
- Express Stores

## Tech Stack

- React
- TypeScript
- Electron
- SQLite
- Tailwind CSS

## Development Method

Build one scrum at a time.
No future scrum may be started until the previous scrum is fully complete and tested.
Each scrum must leave the application in a working state.
