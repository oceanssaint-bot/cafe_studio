# Scrum 01

## Goal
Create working desktop application shell.

## Requirements

Build:
- Electron
- React
- TypeScript
- SQLite

Layout:

Sidebar
- Dashboard
- Month End
- Stores
- Reports
- Settings

Pages may contain placeholder content.

## Success Criteria

- Application launches
- Navigation works
- SQLite database initializes
- No runtime errors

## Do Not Build

- Reports
- Forms
- Automation
- Calculations

Only foundation.
